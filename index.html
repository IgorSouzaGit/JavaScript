Criar(createElement) e Inserir(appendChild) elementos no HTML

var versa = document.createElement("p"); versa.innerHTML = "versa"; var sentra = document.createElement("p"); sentra.innerHTML = "sentra"; document.getElementById("nissan").appendChild(versa); document.getElementById("nissan").appendChild(sentra);

Remover elementos do HTML (removeChild)

var versa = document.createElement("p"); versa.innerHTML = "versa"; var sentra = document.createElement("p"); sentra.innerHTML = "sentra"; document.getElementById("nissan").appendChild(versa); document.getElementById("nissan").appendChild(sentra); document.getElementById("nissan").removeChild(versa);
